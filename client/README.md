# Got Todo Client

## Project setup

1. install packages
```
pnpm install
```

2. local develop
需求 `.env.development` file
```
pnpm dev
```

3. lint
```
pnpm lint
```

4. build dist(production)
需求 `.env` file
```
pnpm build
```

5. 本地啟動專案(production)並 host on 8080 port
需求 `.env` file
```
pnpm run docker:front
```
